# Educase Assignment

A React + TypeScript implementation of the PopX mobile app design provided in the assignment.

## Overview

This project recreates the given Adobe XD design as a pixel-accurate mobile interface centered on the webpage. The application includes navigation between all required screens and reusable UI components built with React.

## Features

- Landing Page
- Signup Page
- Login Page
- Account Settings Page
- React Router based navigation
- Reusable components
  - Button
  - Input
  - Radio Group
- TypeScript support
- Tailwind CSS styling

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
